### Steam-Idler
Steam-Idler is a simple tool for emulating activity on the Steam platform. 
It allows users to automatically launch selected games and leave them offline to show other users that they are in the game. 
Steam-Idler is very easy to use and does not require special programming skills or knowledge. 
This is a useful tool for users who want to keep active on their Steam profile without having to actually play games.

### How to use:

```
[1] Install NodeJS https://nodejs.org/
```
```
[2] Download this script (Code -> Download ZIP). Unpack the archive.
```
```
[3] Use the console to open the unpacked folder!
    Enter in the console: npm i
```
```
[3*] If for some reason the script does not work for you, try entering this 
     command into the console: npm audit fix
```
```
[4] Configure the index.js file for your account.
```
```
[5] Run the script - enter in the console: node index.js
```
