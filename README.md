### Steam-Idler

Steam-Idler is a user-friendly tool designed to simulate activity on the Steam platform, allowing users to automatically launch selected games and appear offline. This tool is ideal for those who wish to maintain an active presence on their Steam profile without actively playing games. No special programming skills or extensive knowledge are required to use Steam-Idler.

### How to Use:

**1. Install NodeJS**
   - Download and install NodeJS from [https://nodejs.org/](https://nodejs.org/).

**2. Download the Script**
   - Download the script by clicking on "Code" and selecting "Download ZIP." Unpack the archive.

**3. Navigate to the Unpacked Folder**
   - Open the console and navigate to the unpacked folder using the `cd` command.

**4. Install Dependencies**
   - Enter the following command in the console: `npm i`

   **4* If Issues Arise**
   - If the script encounters issues, attempt to resolve them by entering: `npm audit fix`

**5. Configure `index.js`**
   - Open the `index.js` file and configure it with your Steam account details.

**6. Run the Script**
   - Execute the script by entering the following command in the console: `node index.js`

Enjoy using Steam-Idler to effortlessly maintain activity on your Steam profile without the need to actively play games!
